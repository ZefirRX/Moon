## ReadMe
Для запуска проектом понадобится среда разработки QT Сreater.
Иерархия
1. Server - консольный проект QT. Содержит сервер для обмена сообщениями через локальную сеть. (Ловит и обрабатывает их)
2. Client+Logins - Qt проект с визуальным интерфейсом для клиентов, а также с экраноми авторизации и регистрации
3. Document - содержит Pitch в котором расписана главная идея проекта а так же WireFrame и User Flow
